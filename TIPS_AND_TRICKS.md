# TrackFlow Tips & Tricks

A collection of obscure features, power user tips, and hidden functionality in TrackFlow.

## üéØ Workflow & Progress Features

### Auto-completion of New Workflow Steps

When you add a new step to your workflow, TrackFlow automatically marks it as complete for songs that already had all previous steps finished. This means if you're 100% done with a song and add a new step, it won't suddenly show as incomplete.

### Workflow Inheritance for Collaborators

When you collaborate on someone else's song, you automatically use the song owner's workflow, not your own. This ensures everyone is aligned on what needs to be done.

### Reset to Default Workflow

If you've customized your workflow too much, you can reset it to the default template from the workflow settings page.

### Pack Priority System

Packs have a priority field (1-5 scale, where 5 is highest). You can set priorities to organize your workflow and sort by them.

### Optional Songs Don't Count Toward Completion

Mark songs as "optional" to exclude them from pack completion calculations. Perfect songs you're unsure about including.

### Cross-Pack Bulk Selection

You can select songs from multiple different packs simultaneously for bulk operations. This works on the Future Plans and Released pages.

### Pack-Level Selection

Use the pack header checkbox to select all songs in a pack at once for quick bulk operations.

## üéµ Song Management Hacks

### In-Place Editing with Auto-Save

Click any song field directly in the table to edit it. Changes save automatically when you click away or press Enter.

### Smart Title Cleaning

The system has an advanced remaster tag cleaning function that removes patterns like "(Remastered 2010)", "(Deluxe Edition)", "- 2010 Remaster", and many other variations. Use the "Clean Remaster Tags" bulk action to clean up your entire library.

### DLC Status Checking

The system automatically checks and warns you if songs are official Rock Band DLC when you create them.

### File Links for Finished Songs

When a song is 100% complete (all authoring fields done), you can add file links without requiring a message. For in-progress songs, a message is required.

### Automatic Enhancement Toggle

You can disable automatic Spotify metadata fetching in User Settings. When disabled, new songs won't auto-enhance, but you can still manually enhance them.

### Playlist Import Status Selection

When importing a Spotify playlist, you can choose the initial status (Future Plans, In Progress, or Released) for all imported songs.

### Smart Metadata Normalization

Spotify enhancement uses advanced title normalization that strips accents, converts special characters, and handles various punctuation styles for better matching.

### Manual Enhancement with Multiple Results

When manually enhancing a song, you can choose from multiple Spotify matches if the automatic match isn't perfect.

### Preexisting Track Marking

In the Album Series editor, you can mark tracks as "preexisting" (already done as customs) or "irrelevant" (interludes, skits, etc.). These flags persist and help you track what still needs to be done.

### Double Album Series

There's a feature to create "double album series" for packs that have enough tracks from more than one album. Double Album Series are generally considered as one pack and released together.

### Create from Pack

If you have 4 or more songs from one album in a pack, You can convert an existing pack directly into an Album Series entry, preserving all songs and metadata. Find "Make Album Series" in the pack dropdown.

### Song-Level vs Pack-Level Collaboration

You can collaborate on individual songs without sharing the entire pack. Perfect for specialists who only work on specific tracks.

### View-Only Collaborators

Set collaborators to "view-only" access if you want them to see progress but not make changes.

### Collaboration Search

When searching songs, you can search by collaborator username. The search includes songs where you're a collaborator.

### Bulk Edit Owner

You can change the owner of multiple songs at once using bulk edit. Useful for transferring songs between accounts or correcting ownership.

### Bulk Edit Collaborations

Add or modify collaborations for multiple songs simultaneously through bulk edit.

### Bulk Clean Remaster Tags

Clean up remaster tags, edition suffixes, and year suffixes from multiple songs at once. The cleaning function handles dozens of different patterns.

### Select All Across Pages

The "Select All" button selects all visible songs, but you can combine this with filtering to select specific subsets for bulk operations.

### Achievement Progress Tracking

View your progress toward count-based achievements (like "10 songs released") on the achievements page. This shows how close you are to the next milestone.

### Achievement Categories

Achievements are organized into categories: milestone, activity, quality, social, and special. Filter by category to see what you're missing.

### Achievement Rarity Levels

Achievements have rarity levels: common, uncommon, rare, epic, and legendary. Legendary achievements are the hardest to earn.

### Points Leaderboard

Check the leaderboard to see how your achievement points compare to other users. Your rank is highlighted.

### Year Distribution Charts

The stats page includes interactive charts showing your song distribution by release year. Great for visualizing your music taste evolution.

### Top Collaborators Tracking

The stats page tracks who you've collaborated with most, helping you identify your frequent partners.

### Login Streak Tracking

Your login streak is tracked in user stats. Log in daily to maintain your streak!

### Activity Logging

Many actions are logged to an activity feed (visible to admins). This includes login, song creation, status changes, Spotify imports, and more.

### Mark All as Read

You can mark all notifications as read at once from the notifications page.

### Notification Context Links

Notifications include links to related content (achievements, feature requests, comments) for easy navigation.

### Collapse/Expand All Groups

Use the "Collapse All" / "Expand All" button to quickly collapse or expand all pack groups on a page.

### Sort by Multiple Fields

You can sort songs by title, artist, album, pack, year, or status. Click the sort button again to reverse the order.

### Progress Bar Colors

Progress bars change color based on completion percentage, providing visual feedback at a glance.

### Feature Request Voting

You can upvote or downvote feature requests. Each user gets one vote per request, and you can change your vote.

### Nested Comments

Feature request comments support replies, creating threaded discussions. Replies are indented under parent comments.

### Feature Request Status Tracking

Feature requests can be marked as "done" (implemented), "rejected" (won't be implemented), or open. Rejected requests include a reason from admins.

### Comment Editing

You can edit your feature request comments after posting. Edited comments are marked as "edited".

### Comment Deletion

You can delete your own comments. Deleted comments are marked but the content is hidden.

### Bug Report System

There's a dedicated bug report page where you can report issues. Admins can track and manage these reports.

### Custom Step Names

You can create completely custom workflow steps beyond the predefined ones. Name them whatever makes sense for your process.

### Drag and Drop Reordering

In the workflow settings, you can drag and drop steps to reorder them. The order matters for progress calculations.

### Workflow Templates

The system includes workflow templates that you can use as starting points, or you can create completely custom workflows from scratch.

### Step Display Names

Each workflow step has both a technical name (like "demucs") and a display name (like "Demucs"). You can customize the display name without affecting functionality.

### Preferred Contact Method

Set your preferred contact method (email or Discord) and Discord username for collaboration purposes. This will be visible to other users.

### Album Cover Art

Album covers are automatically fetched from Spotify and displayed throughout the interface. You can manually update cover art if needed.

### Celebration Animations

When you complete songs, you get celebration animations (fireworks) to make the moment special!

### Case-Insensitive Search

All searches are case-insensitive, so you don't need to worry about capitalization.

### Partial Matching

Search uses partial matching, so searching "beat" will find "The Beatles" and "Beat It".

### Help Page Sections

The help page is organized into sections: Getting Started, Songs & Packs, Statuses & Workflow, Collaboration, Spotify Features, Bulk Operations, Album Series, and FAQ.

### Contextual Help

Many features include tooltips or help text explaining what they do and how to use them.
